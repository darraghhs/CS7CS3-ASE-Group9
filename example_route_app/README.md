## Example app to demonstrate the google route and geocoding apis as well as FirebaseDB integration

### Steps to run app
1. Run `python3 -m venv venv`
2. Run `source venv/bin/activate`
3. Run `pip3 install -r requirements.txt`
4. Before running the app, a firebase authentication details json needs to be present
5. To run app on localhost run `flask run`

Run `deactivate` to deactivate the venv
